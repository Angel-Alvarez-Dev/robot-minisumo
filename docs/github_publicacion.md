## Recomendacion

- Mantener estos videos en el repositorio porque son ligeros y documentan el prototipo funcional.
- Si en el futuro se agregan videos mayores a 50 MB, considerar comprimirlos o publicarlos como release asset.
- Si un archivo supera 100 MB, GitHub puede rechazar el push; en ese caso no hacer commit del video y documentar un enlace externo o usar GitHub Releases.
- Antes de publicar el repositorio, revisar que fotos y videos no incluyan datos personales, rostros, ubicaciones o informacion no deseada en el fondo.

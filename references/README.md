# Referencias tecnicas

Carpeta minima de referencias de componentes.

Se conservan solo archivos necesarios para documentar y validar el proyecto:

- Datasheets relevantes.
- Pinouts.
- Imagenes de dimensiones.
- Esquematico de la shield.

No se conservan en la estructura limpia los paquetes CAD completos, renders duplicados ni archivos temporales de modelado.

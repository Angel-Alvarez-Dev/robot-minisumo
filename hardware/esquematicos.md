# Esquematicos

El esquematico KiCad representa el cableado real sobre la Arduino Nano Expansion I/O Shield.

## Bloques

- Arduino Nano + Shield.
- Alimentacion.
- HC-SR04.
- TCRT5000 Left, Right y Back.
- SG90 Left y Right.
- KY-012.
- GND comun.

## Notas

- D0/D1 quedan reservados para RX/TX Serial.
- Verificar corriente de servos SG90.
- La Shield facilita conexion, pero no aumenta la capacidad de corriente.
